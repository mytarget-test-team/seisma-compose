Test analytic system
====================

Installation
------------

```bash
curl -L https://raw.githubusercontent.com/trifonovmixail/seisma-compose/master/install | sudo bash -
```

Just open localhost and enjoy!


Requirements
------------

* Docker
* Python 2.7 or more
* GNU make
* curl


OS supported
------------

* CentOS 7
* Ubuntu 16
* Debian with systemd
